# **App Name**: ChronoVault OS

## Core Features:

- Capsule Creation & Management: Create new time capsules with a title, content, and type (Message, Goal, Memory, Secret). Set a specific unlock date and time. Edit or delete capsules before their designated lock time.
- Locked & Unlocked Capsules: Capsules remain inaccessible until their scheduled unlock date and time. Displays a live countdown to unlocking. Content is revealed upon successful unlock.
- Timeline & Future Preview: View a chronological timeline of past, present (locked), and future (unlocked) capsules. Includes a 'future unlock preview' to see upcoming content dates.
- Tags & Moods: Enhance capsules with custom tags (e.g., #school, #life) and expressive emoji mood indicators for improved organization and emotional context.
- Search & Filter: Quickly locate specific capsules using a global search bar. Filter capsules by type, date range, associated tags, or recorded mood.
- Writing Streak: Motivate consistent use by tracking a daily 'writing streak' based on the creation of new capsules.
- Security Lock: Secure the application with a simple PIN-based lock mechanism to protect personal data from unauthorized access.

## Style Guidelines:

- The primary color, chosen for its sophisticated digital feel, is a deep violet-blue, like #6633CC (HSL: 250, 60%, 50%).
- The background features a subtle, very dark purple-blue, providing depth for the 'futuristic OS' aesthetic: #18151C (HSL: 250, 15%, 10%).
- A vibrant sky blue serves as the accent color, chosen for its clarity and contrast, illuminating interactive elements and key information: #5FAECF (HSL: 220, 80%, 65%).
- Headline and body font: 'Space Grotesk', a proportional sans-serif, delivers a distinct computerized, techy, and scientific aesthetic perfect for a futuristic OS.
- Utilize crisp, monochromatic outline icons with consistent stroke weights to reinforce the clean and modern digital interface, suitable for glassmorphism.
- Prioritize a mobile-first, card-based layout with generous padding and large touch targets. Implement a bottom navigation bar with a Floating Action Button (FAB) for core actions, ensuring thumb-friendly usability. Responsive design is secondary but essential for larger screens.
- Incorporate subtle and smooth transitions for state changes, navigation, and content revealing. Apply depth-enhancing animations to UI elements, consistent with a modern glassmorphism style, ensuring a fluid and responsive user experience.
